The Dunder Mifflin Company


The Dunder Mifflin Network

This is a scenario of a company named Dunder Mifflin. This is the fictional paper company from the TV show 'The Office'. I am using the names of the characters from the show as my users, and I am the System Administrator and a network of virtual machines to represent the hypothetical computer network of the company.




The figure above shows the Dunder-Mifflin network. The network consists of six production machines that are initially named Machine A through Machine F. The network consists of a connection to the ISP (WAN), a Demilitarized Zone (DMZ) network and an internal network (LAN). I will configure the machines to perform specific tasks such as being a web server, DNS server and a file server. The articles on how I configured them can be found here.

Machine A 100.64.0.26 (Rocky 9.2)
This is the gateway machine and acts as a router to connect the various networks (WAN, LAN and DMZ). This is also a DHCP Server. You can find how I configured this as a DHCP server here.

Machine E 10.21.32.2 (Rocky 9.2)
This is the file server machine on the internal network. You can find how I configured the file server here. The machine is not directly accessible to the outside world and can only be reached by pivoting through Machine A. It contains the home directories of all the users that can be accessed by other machines on the LAN which we don't show explicitly.

Machine B 100.64.26.2 (Rocky 9.2)
This is a DNS Server. The article on how I configured this will be up soon.

Machine C 100.64.26.3 (Debian 12.1)
This is a Web Server. The article on how I configured this will be up soon.

Machine D 100.64.26.4 (Rocky 9.2)
This is a Backup Web Server. The article on how I configured this will be up soon.

Machine F 100.64.26.2 (Debian 12.1)
This is a Backup DNS Server. The article on how I configured this will be up soon.
Machines B,C,D,F are on the DMZ network (100.64.26.0/24). They are accessible from the internet through Machine A.
All the machines run an ssh server.
